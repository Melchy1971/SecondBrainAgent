### Spalten‑Beschreibung & Tipps

| Spalte | Zweck & Hinweise |
| --- | --- |
| **Kategorie** | Erleichtert das Filtern (z. B. „Privat“, „Arbeit“, „Versicherung“). |
| **Plattform / Service** | Name der Website oder App (z. B. Amazon, Allianz‑Onlineportal). |
| **Benutzername / E‑Mail** | Anmeldekennung. |
| **Passwort** | Trage _keine_ Klartext‑Passwörter ein. Nutze <br>• einen Passwort‑Manager (KeePass, Bitwarden, 1Password …) und verlinke nur den Eintrag, oder <br>• einen Hinweis („siehe Manager“, „lokal abgelegt“). |
| **Versicherungsgesellschaft** | Falls es sich um ein Versicherungsportal handelt. |
| **Vertrags‑ / Policen‑Nr.** | Schneller Zugriff bei Schadenmeldungen. |
| **Kontakt** | Support‑URL, Hotline oder E‑Mail. |
| **2‑Faktor aktiv?** | „Ja/Nein“ – so erkennst du Schwachstellen und kannst 2FA nachrüsten. |
| **Notizen** | Zusätzliche Infos (z. B. Kündigungsfrist, letzter Passwortwechsel). |

#### Sicherheitsempfehlungen

1. **Passwort‑Manager statt Klartext**  
   Speichere Passwörter verschlüsselt (lokal oder in der Cloud, je nach Risiko‑Appetit).  
2. **Starke, einzigartige Passwörter**  
   Generiere pro Dienst ein langes, zufälliges Passwort.  
3. **Aktiviere 2‑Faktor‑Authentifizierung (2FA)**  
   Nutze Authenticator‑Apps oder Hardware‑Tokens.  
4. **Regelmäßige Überprüfung**  
   Plane einen halbjährlichen Check, um alte Konten zu schließen und Passwörter zu rotieren.  
5. **Backup & Verschlüsselung**  
   Halte mindestens ein verschlüsseltes Backup deines Passwort‑Tresors bereit.

Viel Erfolg beim Ausfüllen! Wenn du später Anpassungen oder weitere Spalten brauchst, sag einfach Bescheid.
