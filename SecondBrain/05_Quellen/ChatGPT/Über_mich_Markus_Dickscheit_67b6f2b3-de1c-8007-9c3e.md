## 1. Beiträge manuell im Voraus planen (einfachste Variante)

WordPress verfügt über eine integrierte Funktion, mit der du Blogbeiträge planen kannst. Damit kannst du bereits heute mehrere Beiträge vorschreiben und jeweils das Veröffentlichungsdatum (z. B. 09:00 Uhr am folgenden Tag) festlegen. So stellst du sicher, dass jeden Tag automatisch ein Beitrag live geht.

### Schritt-für-Schritt-Anleitung:
1. **Neuen Beitrag erstellen**  
   Gehe in deinem WordPress-Dashboard zu **Beiträge → Erstellen** (oder „Add New“).
2. **Inhalt verfassen**  
   Schreibe deinen Beitrag zum gewünschten Thema (z. B. „Tischtennis – Der perfekte Aufschlag“).
3. **Veröffentlichungsdatum einstellen**  
   - Rechts oben im Editor (bei modernen WP-Versionen im Gutenberg-Editor) findest du den Abschnitt „Veröffentlichen“.  
   - Klicke auf das Datum bzw. die Uhrzeit und wähle dort **morgen**, **09:00 Uhr** aus.  
4. **Planen**  
   Klicke auf **Planen** (statt „Veröffentlichen“). Dein Beitrag wird jetzt automatisch zum festgelegten Zeitpunkt erscheinen.

Diese Vorgehensweise kannst du für jeden Beitrag separat wiederholen. Wenn du zum Beispiel drei Beiträge pro Tag (Tischtennis, ChatGPT, 3D-Druck) benötigst, legst du einfach drei Entwürfe an und planst sie jeweils für 09:00 Uhr an aufeinanderfolgenden Tagen.

Vorteil:  
- Einfache Umsetzung, kein Code erforderlich.  
- Du hast die volle Kontrolle über die Inhalte und das genaue Timing.  

Nachteil:  
- Du musst die Beiträge manuell verfassen und planen. Das wird auf Dauer mühsam, wenn du wirklich *täglich* und ohne Pause veröffentlichen willst.

## 2. Automatisierung per WP-Cron und Plugin (fortgeschrittene Variante)

Wenn du keine Lust hast, jeden Tag manuell Entwürfe zu erstellen, kannst du mithilfe eines kleinen Plugins und WordPress-Cronjobs das Erstellen/Veröffentlichen automatisieren. Im Folgenden ein Beispiel, wie so ein Plugin aussehen könnte. Es legt einmal täglich (um 09:00 Uhr) automatisch drei Beiträge an – jeweils zu Tischtennis, ChatGPT und zur Creality Ender 3 Max Neo.

> **Hinweis**: Um genau 09:00 Uhr zu treffen, sollte dein WordPress-Cron korrekt eingerichtet sein. WordPress-Cron läuft standardmäßig nur bei Seitenaufrufen. Bei Hosting-Anbietern wie IONOS, Strato, etc. kannst du einen echten Cronjob anlegen, der z. B. alle 15 Minuten `wp-cron.php` aufruft und somit die geplanten Tasks abarbeitet.  

### Schritt-für-Schritt-Anleitung

#### 1. Neues Plugin anlegen

Erstelle in deinem WordPress-Installationsverzeichnis unter `wp-content/plugins/` einen Ordner, z. B. `auto-daily-posts/`, und darin eine Datei namens `auto-daily-posts.php` mit folgendem Inhalt:

```php
<?php
/*
Plugin Name: Auto Daily Posts
Description: Erstellt täglich um 09:00 Uhr automatisch drei Blogbeiträge zu Tischtennis, ChatGPT und Creality Ender 3 Max Neo.
Version: 1.0
Author: Dein Name
*/

// Verhindert direkten Zugriff
if ( ! defined( 'WPINC' ) ) {
    die;
}

/**
 * 1. Custom Cron-Interval hinzufügen (1 Tag).
 */
function adp_custom_cron_schedules( $schedules ) {
    // Falls du mehrere Intervalle benötigst, hier anpassen. 24 Stunden = 86400 Sekunden.
    if(!isset($schedules['daily_9_am'])) {
        $schedules['daily_9_am'] = array(
            'interval' => 86400, // 1 Tag
            'display'  => __( 'Einmal pro Tag um 09:00 Uhr', 'auto-daily-posts' )
        );
    }
    return $schedules;
}
add_filter( 'cron_schedules', 'adp_custom_cron_schedules' );

/**
 * 2. Aktivierungshaken: Cron-Event nur einmal beim Aktivieren des Plugins planen.
 */
function adp_activate_plugin() {
    // Prüfen, ob Event schon geplant ist
    if ( ! wp_next_scheduled( 'adp_create_daily_posts' ) ) {
        // Datum für den ersten Lauf morgen um 09:00 Uhr berechnen
        $timestamp = adp_get_next_9_am_timestamp();
        wp_schedule_event( $timestamp, 'daily_9_am', 'adp_create_daily_posts' );
    }
}
register_activation_hook( __FILE__, 'adp_activate_plugin' );

/**
 * 3. Deaktivierungshaken: Cron-Event entfernen.
 */
function adp_deactivate_plugin() {
    $timestamp = wp_next_scheduled( 'adp_create_daily_posts' );
    if ( $timestamp ) {
        wp_unschedule_event( $timestamp, 'adp_create_daily_posts' );
    }
}
register_deactivation_hook( __FILE__, 'adp_deactivate_plugin' );

/**
 * 4. Funktion zum Erzeugen der Beiträge.
 */
add_action( 'adp_create_daily_posts', 'adp_create_three_posts' );
function adp_create_three_posts() {

    // Heutiges Datum für den Beitragstitel, optional
    $date = current_time( 'Y-m-d' ); 

    // Array mit den Themen
    $topics = array(
        array(
            'title'   => 'Tischtennis – Daily Post ' . $date,
            'content' => 'Hier entsteht ein automatischer Beitrag zum Thema Tischtennis. Viel Spaß beim Lesen!',
        ),
        array(
            'title'   => 'ChatGPT – Daily Post ' . $date,
            'content' => 'Dieser Beitrag beschäftigt sich mit ChatGPT und aktuellen KI-Trends.',
        ),
        array(
            'title'   => 'Creality Ender 3 Max Neo – Daily Post ' . $date,
            'content' => 'Hier findest du Tipps und Tricks rund um den Creality Ender 3 Max Neo und 3D-Druck.',
        ),
    );

    foreach ( $topics as $topic ) {
        // Überprüfen, ob ein ähnlicher Beitrag bereits existiert
        $existing_post = get_page_by_title( $topic['title'], OBJECT, 'post' );

        // Nur erstellen, wenn kein Beitrag mit demselben Titel gefunden wurde
        if ( ! $existing_post ) {
            $new_post = array(
                'post_title'   => wp_strip_all_tags( $topic['title'] ),
                'post_content' => $topic['content'],
                'post_status'  => 'publish',      // Direkt veröffentlichen
                'post_author'  => 1,              // Autor (ID des Admins oder gewünschten Users)
                'post_type'    => 'post',
            );

            // Beitrag erstellen
            wp_insert_post( $new_post );
        }
    }

}

/**
 * 5. Helper-Funktion, um 09:00 Uhr vom nächsten Tag zu bekommen
 */
function adp_get_next_9_am_timestamp() {
    // Aktuelle Zeit in WordPress-Zeitstempel
    $now = current_time('timestamp');

    // Morgen 09:00 Uhr
    $tomorrow_9am = strtotime('tomorrow 09:00', $now);

    return $tomorrow_9am;
}
```

**Was macht dieses Plugin?**
1. Es definiert ein eigenes Cron-Intervall (24 Stunden).
2. Bei der Aktivierung des Plugins wird ein Event angesetzt, das morgen um 09:00 Uhr startet. Danach läuft es jeden Tag um 09:00 Uhr erneut.
3. Die Funktion `adp_create_three_posts()` erstellt drei neue Beiträge zu den Themen Tischtennis, ChatGPT und Ender 3 Max Neo.
4. Die Beiträge werden jeweils direkt veröffentlicht (Status `publish`).

#### 2. Plugin hochladen und aktivieren
- Lade den Ordner `auto-daily-posts` (mit der `auto-daily-posts.php` Datei) in dein WordPress-Verzeichnis unter `wp-content/plugins`.
- Gehe ins Dashboard: **Plugins → Installierte Plugins**, suche nach „Auto Daily Posts“ und aktiviere es.

#### 3. WordPress-Cron sicherstellen
- Standardmäßig führt WordPress Cron-Jobs nur aus, wenn jemand eine Seite deiner Website besucht. 
- Bei stark besuchten Seiten reicht das oft. Andernfalls richte beim Hoster einen echten Cronjob ein, der z. B. alle 15 Minuten `wp-cron.php` aufruft. 
- So stellst du sicher, dass der tägliche Post wirklich punktgenau um 09:00 Uhr veröffentlicht wird.

Vorteil:  
- Du musst nichts mehr manuell machen, jeden Tag werden automatisch drei Beiträge erstellt.  
- Du kannst die Inhalte im Plugin-Code beliebig anpassen (z. B. jeden Tag andere Titel, Texte, Kategorien usw.).  

Nachteil:  
- Technisch etwas aufwendiger, du musst Code anpassen und Verständnis für WP-Cron haben.  
- Deine Beiträge sind nicht individuell formuliert, es sei denn, du integrierst eine Logik (z. B. Zitate, dynamische Inhalte etc.).

### 10. Assistent

**3D-Druck im privaten Umfeld: Eine Revolution in den eigenen vier Wänden**

3D-Druck zählt längst nicht mehr nur zu den Hightech-Verfahren großer Industrieunternehmen. Immer mehr Privatpersonen entdecken die Faszination der additiven Fertigung für sich – und das aus gutem Grund. Die Möglichkeiten, eigene Ideen, Ersatzteile und Dekorationen schnell und kostengünstig herzustellen, wachsen stetig. In diesem Beitrag erfährst du, warum 3D-Druck auch im heimischen Umfeld ein spannendes Thema ist, was du für den Einstieg benötigst und welche Einsatzbereiche sich besonders anbieten.

---

### 1. Was ist 3D-Druck?
3D-Druck (oder additive Fertigung) ist ein Verfahren, bei dem ein Objekt Schicht für Schicht aus einem Material aufgebaut wird. In den meisten privaten Haushalten kommen sogenannte **FDM-Drucker** (Fused Deposition Modeling) zum Einsatz. Hierbei wird ein Kunststoff-Filament (z. B. PLA oder PETG) durch eine erhitzte Düse geschmolzen und präzise Schicht für Schicht aufgetragen, bis das gewünschte Objekt entsteht.  
Weitere Verfahren (etwa Resin-Druck per SLA/DLP) sind ebenfalls für Heimanwender verfügbar, erfordern aber oft etwas mehr Aufwand und Erfahrung.

---

### 2. Warum 3D-Druck zu Hause?
- **Kreative Freiheit:** Die Möglichkeit, eigene Ideen oder digitale Vorlagen aus dem Internet in reale Objekte umzusetzen, eröffnet unendliche Gestaltungsspielräume.  
- **Kostenersparnis:** Statt teure Fertigprodukte zu kaufen, können viele Alltagsgegenstände – wie Handyhalterungen, Ersatzteile oder Kabelhalter – einfach selbst gedruckt werden.  
- **Lerneffekt:** Der Umgang mit 3D-Druck fördert technisches Verständnis, Design-Kenntnisse und Problemlösungskompetenzen. Besonders Kinder und Jugendliche lernen spielerisch grundlegende MINT-Fähigkeiten.  
- **Individualisierung:** Wer etwas Besonderes sucht, kann sein Objekt genau nach den eigenen Vorstellungen anpassen (z. B. persönliche Gravuren, Abmessungen, Farben).  

---

### 3. Voraussetzungen und Hardware für den Heimgebrauch
- **3D-Drucker:** Für Einsteiger eignen sich Modelle wie der Creality Ender 3 oder andere preiswerte FDM-Drucker. Diese bieten ein gutes Preis-Leistungs-Verhältnis und eine aktive Community für Hilfe und Austausch.  
- **Filament:** Das gebräuchlichste Material im Heimbereich ist PLA (Polylactid), ein biologisch abbaubarer Kunststoff. PETG, ABS oder flexible Filamente eröffnen darüber hinaus weitere Einsatzmöglichkeiten.  
- **PC und Slicing-Software:** Für die Umwandlung eines 3D-Modells (z. B. STL-Datei) in Druckbefehle benötigt man ein Slicer-Programm wie Cura oder PrusaSlicer. Diese Software bereitet das Modell für den jeweiligen Drucker auf.  
- **Bau- und Bastelbereitschaft:** Gerade bei günstigen Druckern ist gelegentliches Justieren oder kleinere Upgrades normal. Wer Freude am Basteln und Tüfteln hat, ist hier klar im Vorteil.

---

### 4. Woher kommen die Druckvorlagen?
- **3D-Modell-Bibliotheken:** Plattformen wie Thingiverse, Printables oder MyMiniFactory bieten unzählige kostenlose oder kostenpflichtige 3D-Modelle zum Download. Von nützlichen Alltagsgegenständen über Dekoration bis hin zu komplexen Kunstwerken ist für jeden Geschmack etwas dabei.  
- **Eigene Konstruktion:** Mithilfe von CAD-Programmen (z. B. Tinkercad, Fusion 360, Blender) können NutzerInnen ihre eigenen Designs erstellen. Dabei sind Kreativität und ein wenig Übung gefragt, um individuelle Modelle zu realisieren.

---

### 5. Spannende Einsatzbereiche zu Hause
1. **Ersatzteile und Reparaturen:** Beschädigte Plastikgriffe oder Halterungen im Haushalt lassen sich oft schnell nachdrucken. So kann man defekte Geräte reparieren, statt sie wegzuwerfen.  
2. **Deko und Personalisierung:** Ob Vasen, Figuren oder individualisierte Geschenke – mit 3D-Druck wird das Zuhause persönlicher und kunstvoller.  
3. **Lernprojekte:** Bastelprojekte für Kinder, naturwissenschaftliche Modelle oder Bauteile für Robotik-Projekte – der pädagogische Mehrwert ist enorm.  
4. **Modellbau und Hobby:** Für Hobbybereiche wie RC-Modelle, Tabletop-Spiele oder Architektur lassen sich passgenaue Teile drucken.

---

### 6. Herausforderungen und Tipps
- **Kalibrierung und Wartung:** Ein sauber eingestellter Drucker ist das A und O für hochwertige Drucke. Regelmäßige Wartung (z. B. Düsenreinigung, Filamentwechsel) beugt vielen Problemen vor.  
- **Sicherheit:** Auch wenn 3D-Drucker für den Hausgebrauch konstruiert sind, sollte man auf eine gute Belüftung achten. Einige Filamente können beim Druck Gerüche oder Partikel freisetzen.  
- **Geduld lernen:** Der Druck eines größeren Objekts kann mehrere Stunden dauern. Dazu kann es immer mal wieder Fehldrucke geben – vor allem zu Beginn. Ein wenig Frustrationstoleranz hilft, am Ball zu bleiben.  
- **Community-Ressourcen nutzen:** Die 3D-Druck-Szene ist äußerst hilfsbereit. Bei Problemen helfen oft Foren, YouTube-Tutorials oder Facebook-Gruppen.  

---

### Fazit
3D-Druck im privaten Umfeld bietet eine faszinierende Möglichkeit, Technik hautnah zu erleben, kreative Ideen umzusetzen und gleichzeitig Alltagsprobleme auf innovative Weise zu lösen. Wer Freude am Ausprobieren und Lernen hat, findet in dieser Technologie ein spannendes Hobby und eine Quelle endloser Inspiration. Ob du einzelne Bauteile oder Kunstobjekte entwerfen möchtest, Modellbau betreibst oder einfach nur Spaß am Basteln hast – der 3D-Druck eröffnet dir eine Welt voller Möglichkeiten. 

Mit einem **bezahlbaren FDM-Drucker**, etwas **Einarbeitung** in die Slicing-Software und ein wenig **Experimentierfreude** steht deinem Einstieg nichts mehr im Weg. Wer weiß, vielleicht hast du schon bald deine eigene kleine „Fabrik“ im Wohnzimmer!
