### 2. Assistent

Hier eine übersichtliche Liste gängiger Outlook-Shortcuts für Windows:

### E-Mail-Verwaltung
- **Strg + N**: Neue E-Mail erstellen  
- **Strg + Umschalt + M**: Neue Nachricht aus jedem Outlook-Fenster starten  
- **Strg + R**: Auf die ausgewählte E-Mail antworten  
- **Strg + Umschalt + R**: Allen antworten  
- **Strg + F**: E-Mail weiterleiten  
- **Alt + S** oder **Strg + Enter**: E-Mail senden

### Navigation & Ansichten
- **Strg + 1**: Wechsel in die E-Mail-Ansicht  
- **Strg + 2**: Kalender-Ansicht öffnen  
- **Strg + 3**: Zu den Kontakten wechseln  
- **Strg + 4**: Aufgaben-Ansicht anzeigen  
- **Strg + 5**: Notizen öffnen  
- **Strg + Y**: Zu einem anderen Ordner navigieren  
- **F9**: Senden/Empfangen aller Konten

### Ordnerverwaltung
- **Strg + Umschalt + I**: Zum Posteingang wechseln  
- **Strg + Umschalt + O**: Zum Postausgang wechseln

### Suche & Markierung
- **Strg + E** oder **F3**: Zum Suchfeld springen  
- **Alt + Q**: Fokus auf das Suchfeld (Schnellsuche)  
- **Strg + Q**: Als gelesen markieren  
- **Strg + U**: Als ungelesen markieren

### Kalender & Termine
- **Alt + N**: Neuen Termin oder Besprechungseinladung erstellen

*Hinweis:* Einige Shortcuts können je nach Outlook-Version oder individuellen Einstellungen variieren. Diese Liste deckt jedoch die gängigsten Funktionen ab.

## 1. Tastenkombinationen in Python-Entwicklungsumgebungen

### a) Python IDLE (Standard-Editor)
- **F5**: Aktuelles Skript ausführen  
- **Strg + N**: Neues Editor-Fenster öffnen  
- **Strg + O**: Datei öffnen  
- **Strg + S**: Datei speichern  
- **Strg + Z**: Rückgängig machen  
- **Strg + Y**: Wiederherstellen (Redo)  
- **Strg + F**: Suchen  
- **Strg + H**: Ersetzen  

> Diese Shortcuts gelten für die Standard-IDLE-Umgebung, die mit Python mitgeliefert wird.

### b) IDE-spezifische Tastenkombinationen

#### In PyCharm (Windows/Linux)
- **Strg + /**: Kommentar ein- bzw. auskommentieren  
- **Strg + Alt + L**: Code formatieren  
- **Strg + D**: Zeile duplizieren  
- **Strg + Y**: Zeile löschen  
- **Shift + Shift**: Überall suchen (Schnellsuche)  
- **Strg + B**: Zur Deklaration springen  
- **Alt + F7**: Verwendungs-Suche starten  

#### In Visual Studio Code (mit Python-Erweiterung)
- **Strg + Shift + P**: Befehlspalette öffnen  
- **F5**: Debugging starten  
- **Strg + `**: Integriertes Terminal öffnen  
- **Shift + Alt + F**: Code formatieren  
- **Strg + /**: Kommentarzeile ein- bzw. ausschalten  
- **Strg + Shift + O**: Zu einem Symbol im Code springen  

> Die konkreten Tastenkombinationen können je nach Betriebssystem und individuellen Einstellungen variieren.

## 2. Python-Sprachidiome als „Code-Shortcuts“

Neben Tastenkombinationen gibt es in Python zahlreiche Sprachkonstrukte, die den Code kompakter und lesbarer machen:

- **List Comprehensions:**  
  ```python
  squares = [x**2 for x in range(10)]
  ```
- **Dictionary Comprehensions:**  
  ```python
  mapping = {x: x**2 for x in range(5)}
  ```
- **Ternärer Operator:**  
  ```python
  result = "Positiv" if number > 0 else "Negativ"
  ```
- **Walrus Operator (Python 3.8+):**  
  ```python
  if (n := len(items)) > 0:
      print(f"{n} Elemente gefunden")
  ```
- **Mehrfachzuweisung & Unpacking:**  
  ```python
  a, b, c = 1, 2, 3
  first, *middle, last = [1, 2, 3, 4, 5]
  ```
- **Slicing-Shortcuts:**  
  ```python
  reversed_list = my_list[::-1]
  ```
- **Lambda-Funktionen:**  
  ```python
  double = lambda x: x * 2
  ```

> Diese Sprachfeatures dienen als "Abkürzungen", um häufige Operationen effizient und übersichtlich auszudrücken.
