### 1. Benutzer

Wie füge ich in Siquando Web Pro 9 im Menü einen Link auf eine andere Homepage ein

### 2. Assistent

In **Siquando Web Pro 9** kannst du im Menü ganz einfach einen Link zu einer **externen Website** (also einer anderen Homepage) einfügen. Dafür nutzt du den **Seitentyp "Link"**. So gehst du vor:

---

### ✅ Anleitung: Externen Link ins Menü einfügen

1. **Siquando Web Pro 9 öffnen**  
   Starte dein Projekt im Siquando Web Pro 9.

2. **Im Seitenbaum auf die Stelle klicken**, an der der Link im Menü erscheinen soll.  
   (Also z. B. auf „Hauptmenü“ oder eine Unterkategorie.)

3. **Neue Seite hinzufügen**  
   Klicke auf **„+ Seite hinzufügen“** oder Rechtsklick > „Neue Seite“.

4. **Seitentyp auswählen**  
   Wähle im Auswahlfenster den Typ **„Link“** aus.

5. **Name der Seite**  
   Gib der Seite einen Namen – z. B. „Zur Partnerseite“ – das ist der Text, der im Menü erscheint.

6. **Link-Adresse eingeben**  
   Gib im Feld „Adresse“ die **komplette URL** der externen Website ein (z. B. `https://www.partnerseite.de`).

7. **Ziel auswählen (optional)**  
   Du kannst einstellen, ob der Link in einem **neuen Fenster/Tab** geöffnet werden soll:
   - Häkchen bei **„In neuem Fenster öffnen“** setzen, wenn gewünscht.

8. **Speichern**  
   Bestätige mit **OK** und veröffentliche ggf. deine Website neu.

---

### 🔁 Ergebnis
Jetzt erscheint im Menü deiner Website ein Eintrag, der beim Anklicken die Besucher auf die andere Homepage weiterleitet.

### 4. Assistent

Um in **Siquando Pro 9** im Menü einen **Link auf eine externe Homepage** einzufügen, gehst du wie folgt vor:

---

### 🔗 **Schritt-für-Schritt-Anleitung: Externer Link im Menü**

1. **Siquando Pro 9 öffnen**  
   Starte dein Projekt in Siquando Pro 9.

2. **Navigiere zum Seitenbaum / Menüstruktur**  
   Auf der linken Seite siehst du deine Website-Struktur (Navigation / Seitenbaum).

3. **Neue Seite hinzufügen**
   - Rechtsklick auf den Menüpunkt, unter dem du den Link einfügen möchtest.
   - Wähle **„Neue Seite hinzufügen“** oder **„Eintrag hinzufügen“**.

4. **Seitentyp „Externer Link“ wählen**
   - Im erscheinenden Dialog wählst du **„Externer Link“** oder **„Link (extern)“** als Seitentyp (der genaue Name kann leicht variieren).
   - Klicke auf **„Weiter“** oder **„OK“**.

5. **Linkdetails eingeben**
   - **Bezeichnung:** Der Name, der im Menü erscheinen soll (z. B. „Partner-Website“).
   - **URL:** Die vollständige Internetadresse (z. B. `https://www.partnerseite.de`)
   - Optional: Wähle aus, ob der Link in einem neuen Fenster / Tab geöffnet werden soll (empfohlen für externe Links).

6. **Speichern und Veröffentlichen**
   - Speichere deine Änderungen.
   - Veröffentliche deine Website, damit der neue Link sichtbar wird.

---

### ✅ **Ergebnis**
Der neue Menüpunkt erscheint jetzt wie jede andere Seite im Menü – beim Klicken wird aber die gewünschte externe Homepage aufgerufen.
