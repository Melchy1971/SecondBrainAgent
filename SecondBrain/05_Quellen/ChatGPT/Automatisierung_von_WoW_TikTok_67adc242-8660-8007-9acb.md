## 3. Optimierung für TikTok

1. **Videoformat und Länge**  
   - Achte auf das **9:16-Hochformat** und eine Dauer von maximal 60–90 Sekunden für die meisten TikTok-Videos. Kürzere Videos (15–30 Sekunden) sind oft besonders erfolgreich, da sie die Zuschauer schnell in den Bann ziehen.  

2. **Hooks und Call-to-Action**  
   - Starte das Video mit einem **starken Hook** (z. B. „Wusstest du schon, dass…?“ oder „Diesen Fehler machen 80 % aller WoW-Spieler…“).  
   - Fordere die Zuschauer am Ende des Videos zur **Interaktion** auf (z. B. „Like und folge für mehr WoW-Tipps!“ oder „Welche Klasse spielst du? Schreib es in die Kommentare!“).

3. **Hashtags und Tags**  
   - Füge relevante Hashtags hinzu, z. B. **#WorldOfWarcraft, #WoW, #WoWGuide, #Blizzard, #Gaming, #MMORPG**, um die Sichtbarkeit zu erhöhen.  
   - Achte auf **Trend-Hashtags** und virale Sounds. Wenn es passt, integriere sie, um eine höhere Reichweite zu erzielen.

4. **Musik und Soundeffekte**  
   - Wähle **lizenzfreie Musik** oder trendige Sounds aus der TikTok-Bibliothek, um dein Video akustisch aufzuwerten.  
   - Experimentiere mit Soundeffekten, um Highlights oder Pointen zu betonen.

## 2. Themen-Recherche

Da Google Alerts oder Community-Foren (z. B. Reddit, Blizzard-Forum) keine einfache, universelle API bieten, kannst du mehrere Ansätze fahren:

- **Manuell** (z. B. einmal täglich in einem Skript/Notion/Google Sheet Ideen sammeln).

### Beispiel: ChatGPT als Ideenlieferant

```python
import openai
import os

openai.api_key = os.getenv("OPENAI_API_KEY")

def get_wow_ideas():
    prompt = """
    Du bist ein erfahrener World of Warcraft-Experte. 
    Nenne mir bitte 3 aktuelle Themen oder Trends, über die man heute 
    einen kurzen TikTok-Clip erstellen könnte. Achte auf Relevanz und Neuigkeitswert.
    """
    response = openai.Completion.create(
        engine="text-davinci-003",  # oder gpt-3.5-turbo (Chat-Modell)
        prompt=prompt,
        max_tokens=150,
        temperature=0.7
    )
    ideas_text = response["choices"][0]["text"].strip()
    return ideas_text
```

Diese Funktion liefert dir jeden Tag 3 grobe Ideen. Du kannst sie weiterverarbeiten.

## 3. Skript-Generierung mit ChatGPT

Sobald du dich für ein Thema entschieden hast (z. B. „Neue Raid-Strategien“), kannst du ChatGPT bitten, ein kurzes Skript oder einen Text für dein TikTok-Video zu generieren.

```python
def generate_tiktok_script(topic):
    prompt = f"""
    Erstelle ein kurzes, prägnantes Skript für ein TikTok-Video (max. 30 Sekunden) 
    zum Thema: '{topic}'. 
    Nutze einen lockeren, aber informativen Ton, damit World of Warcraft-Fans 
    etwas lernen und unterhalten werden.
    """
    response = openai.Completion.create(
        engine="text-davinci-003",
        prompt=prompt,
        max_tokens=200,
        temperature=0.8
    )
    script_text = response["choices"][0]["text"].strip()
    return script_text
```

Jetzt hast du ein kurzes Skript, das du für die Videoerstellung verwenden kannst.

### Variante B: Selenium + Headless-Browser

1. **Selenium Setup**:

```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.common.keys import Keys
import time
import os

def upload_to_tiktok(video_path, caption):
    # Headless Chrome-Optionen
    options = webdriver.ChromeOptions()
    options.add_argument('--headless')            # Headless Mode
    options.add_argument('--disable-gpu')
    # ggf. User-Agent ändern, um Erkennung zu erschweren
    # options.add_argument('--user-agent=Mozilla/5.0 ...')

    driver = webdriver.Chrome(options=options)

    # 1. Zu TikTok gehen und einloggen
    driver.get("https://www.tiktok.com/login")

    # Beispiel (sehr vereinfacht, oft kommt ein QR- oder Captcha):
    time.sleep(5)
    username_field = driver.find_element(By.NAME, "username")
    password_field = driver.find_element(By.NAME, "password")

    username_field.send_keys(os.getenv("TIKTOK_USERNAME"))
    password_field.send_keys(os.getenv("TIKTOK_PASSWORD"))
    password_field.send_keys(Keys.RETURN)

    time.sleep(10)  # Warten auf erfolgreichen Login

    # 2. Zum Upload-Bereich navigieren (kann sich ändern)
    driver.get("https://www.tiktok.com/upload?lang=en")

    time.sleep(5)

    # 3. Video hochladen
    upload_input = driver.find_element(By.XPATH, '//input[@type="file"]')
    upload_input.send_keys(video_path)  # Lokaler Pfad zum Video

    time.sleep(10)  # Warten, bis das Video hochgeladen ist

    # 4. Caption/Hashtags eingeben
    caption_field = driver.find_element(By.XPATH, '//div[@data-e2e="caption-editor"]//p')
    caption_field.click()
    caption_field.send_keys(caption)

    # 5. Video veröffentlichen
    publish_button = driver.find_element(By.XPATH, '//button[@data-e2e="publish-button"]')
    publish_button.click()

    time.sleep(15)  # Warte, bis Upload abgeschlossen

    driver.quit()
```

- **Wichtig**: Dieser Ansatz kann schnell kaputtgehen, wenn TikTok sein Frontend ändert oder zusätzliche Captchas/Verifikationen einschaltet. Es ist also eher ein *Hack* als eine langfristig stabile Lösung.

## 6. Tägliche Ausführung (Scheduling)

Um deinen Bot jeden Tag zu einer bestimmten Uhrzeit auszuführen, kannst du:

- **Cron-Jobs** (unter Linux/Mac)  
- Das Python-Paket **schedule**  
- Oder Windows Task Scheduler  

verwenden. Beispiel mit `schedule`:

```python
import schedule
import time

def daily_tiktok_workflow():
    # 1. Hole Ideen
    ideas = get_wow_ideas()
    # Wähle eine Idee aus oder frage den Nutzer
    selected_topic = ideas.split('\n')[0]  # Einfach erste Zeile nehmen
    
    # 2. Skript generieren
    script = generate_tiktok_script(selected_topic)
    
    # 3. (Optional) Ingame-Clip auswählen oder generieren
    #    Hier nur Platzhalter, z.B. "clip.mp4"
    video_clip_path = "/path/to/clip.mp4"
    
    # 4. Video zusammenbauen (Intro + clip + Outro) -> output.mp4
    #    Hier ggf. FFmpeg-Aufruf
    #    Oder du lässt es so, wenn du bereits ein fertiges Video hast
    
    final_video_path = "/path/to/output.mp4"
    
    # 5. Caption/Hashtags
    caption = f"{script}\n#WorldOfWarcraft #WoW #Gaming #MMORPG"
    
    # 6. Upload (Variante A oder B)
    # A) Kopiere output.mp4 in deinen Zapier-Ordner
    # import shutil
    # shutil.copy(final_video_path, "/path/to/TikTokReady/")
    
    # B) Selenium-Upload
    # upload_to_tiktok(final_video_path, caption)

# Jeden Tag um 12 Uhr
schedule.every().day.at("12:00").do(daily_tiktok_workflow)

while True:
    schedule.run_pending()
    time.sleep(60)
```

Dieses Skript läuft dauerhaft, überprüft jede Minute, ob 12:00 Uhr erreicht ist, und führt dann den `daily_tiktok_workflow()` aus.

## 7. Erweiterungen und Best Practices

1. **Fehlerbehandlung**: Baue Try/Except-Blöcke ein, um zu reagieren, wenn etwas schiefläuft (z. B. kein Login möglich, ChatGPT nicht erreichbar, etc.).  
2. **Logging**: Speichere Logs über den Fortschritt und mögliche Fehlermeldungen (z. B. in einer Datenbank oder einer Log-Datei).  
3. **Analyse & Reporting**: Automatisiere das Auslesen von Performance-Daten (Views, Likes, Kommentare), z. B. über TikTok-Analytics oder manuelles Auslesen mit Selenium.  
4. **Hashtag-Optimierung**: Lasse ChatGPT oder externe Tools Hashtag-Vorschläge generieren.  
5. **Mehrsprachigkeit**: Du könntest mehrere Sprachen abdecken (z. B. Deutsch und Englisch), um mehr Reichweite zu erzielen.  
6. **Gegenseitige Promotion**: Poste dieselben Kurzvideos auf **YouTube Shorts** und **Instagram Reels**, entweder manuell oder per Zapier/IFTTT (beachte ggf. Wasserzeichen und unterschiedliche Musikrichtlinien).

## Fazit

- **Vollautomatisierung** ist möglich, aber bei den Video-Plattformen oft mit Hürden (API-Beschränkungen, Captchas, sich ändernde Frontends) verbunden.  
- **Kombination** aus halbautomatischen und automatischen Schritten (z. B. ChatGPT für Ideen + Skripte, manuelle Videoerstellung, automatisches Upload-Scheduling) ist in der Praxis häufig stabiler.  
- Für einen **Bot**, der komplett selbstständig Inhalte generiert, bearbeitet und hochlädt, solltest du regelmäßige **Qualitätskontrollen** durchführen, um sicherzustellen, dass keine veralteten oder fehlerhaften Inhalte gepostet werden.
