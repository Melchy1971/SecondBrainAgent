## 🔥 1. Die 80%-Standard-Kombi (für Web & General Dev)

**VS Code + Git + Node.js + passende Extensions**

### Setup:
- **VS Code**
- **Git**
- **Node.js**
- Extensions:
  - ESLint
  - Prettier
  - GitLens
  - Docker (falls relevant)

👉 Das ist ideal für:
- Webentwicklung (React, Vue, Next, etc.)
- Fullstack mit Node
- Skripting
- APIs

**Warum gut?**  
Leichtgewichtig, riesiges Ökosystem, schnell startklar.

**Aber Achtung:**  
Wenn du nur installierst und keine saubere Projektstruktur + Linting-Regeln definierst, wird’s Chaos.

---

## 🧠 2. Python-Dev Kombi

### Setup:
- VS Code
- Python (3.11+)
- Python Extension
- venv oder poetry
- Black + Ruff

👉 Stark für:
- Automatisierung
- Data
- KI
- Backend (FastAPI, Django)

**Gegenargument:**  
Viele sagen „VS Code reicht“.  
Aber: Wenn du große Python-Projekte baust, ist **PyCharm** teilweise produktiver.

Frage an dich:  
Geht’s dir um Skripte oder Enterprise-Level-Projekte?
