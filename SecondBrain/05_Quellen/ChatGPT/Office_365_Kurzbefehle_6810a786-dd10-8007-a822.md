### 2 | Python-Entwicklung  

#### 2.1 Interaktives IPython / REPL  
* **Tab** Autovervollständigung & Objekt-Inspektion  
* **↑ / ↓** Durch Befehls-Historie blättern  
* **Ctrl + A / E** Zum Zeilen-Anfang / -Ende springen  
* **Ctrl + D** Interpreter verlassen  
* **Ctrl + C** Laufenden Code abbrechen citeturn6search1  

#### 2.2 Jupyter Notebook / JupyterLab (Command-Mode = Esc)  
* **Shift + Enter** Zelle ausführen & nächste selektieren  
* **Ctrl + Enter** Zelle ausführen, Fokus behalten  
* **Alt + Enter** Zelle ausführen & neue darunter einfügen  
* **A / B** Zelle darüber / darunter einfügen  
* **D D** Zelle löschen  
* **M / Y** Zelle zu Markdown / Code umwandeln  
* **H** Alle Shortcuts anzeigen citeturn5search1  

*(Tipp: In VS Code gilt zusätzlich **F5** zum Debug-Start, **Ctrl + Shift + P** für die Befehlspalette.)*
