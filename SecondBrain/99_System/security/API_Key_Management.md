# API-Key-Management

## Ziel

API-Keys werden nicht im Vault gespeichert.

## Speicherort

```text
SecondBrain-Agent/config/secrets.local.yaml
```

## Vorlage

```text
SecondBrain-Agent/config/secrets.template.yaml
```

## Regeln

- `secrets.local.yaml` niemals teilen.
- Keine API-Keys in Markdown-Notizen schreiben.
- Keine API-Keys in Reports schreiben.
- Keine API-Keys in Logs schreiben.
