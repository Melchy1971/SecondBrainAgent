# Obsidian Plugin Installation

## Build

```powershell
cd H:\SecondBrainAgent\SecondBrain-Agent\obsidian-plugin
npm install
npm run build
```

## Installationsziel

```text
H:\SecondBrainAgent\SecondBrain\.obsidian\plugins\secondbrain-agent
```

Kopieren:
- manifest.json
- main.js
- styles.css

## Voraussetzungen

Lokale API starten:

```powershell
cd H:\SecondBrainAgent\SecondBrain-Agent
python scripts\rest_api.py
```
