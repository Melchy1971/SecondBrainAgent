# v30.31 — Native Command Center

## Inhalt

- Zentraler Katalog für deutsche Schnellbefehle.
- Kommando-Palette für native GUI und Sprache.
- Historie unter `runtime/native/command_history.jsonl`.
- Freigaben unter `runtime/native/command_approvals.jsonl`.
- Favoriten unter `runtime/native/command_favorites.json`.
- Schreibende Aktionen erfordern Bestätigung.

## Enthaltene Kommando-Gruppen

- System
- RAG
- GUI

## Akzeptanz

- Read-only Befehle sind direkt ausführbar.
- Schreibende Befehle erzeugen Approval-Einträge.
- GUI kann als separates Command Center gestartet werden.
