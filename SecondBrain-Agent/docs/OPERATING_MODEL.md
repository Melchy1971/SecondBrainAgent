# Operating Model

## Rollen

### Owner
Betreibt Agent, entscheidet Struktur, prüft Reports.

### Editor
Pflegt Inhalte und Review Queue.

### Viewer
Liest Vault-Inhalte.

## Täglicher Betrieb

1. Inbox befüllen
2. Import ausführen
3. Intelligence Cycle ausführen
4. Daily Briefing prüfen
5. Review Queue prüfen

## Wöchentlicher Betrieb

1. Vault reindexieren
2. Weighted Graph aktualisieren
3. Wissenslücken prüfen
4. Projektmonitor prüfen
5. Backup prüfen
