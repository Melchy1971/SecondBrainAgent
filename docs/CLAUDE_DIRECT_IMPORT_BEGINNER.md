# Claude Direct Import – Anfänger-Anleitung

## 1. Pakete installieren

```powershell
cd H:\SecondBrainAgent\SecondBrain-Agent\mcp-server
pip install -r requirements.txt
```

## 2. Claude Config öffnen

```powershell
notepad $env:APPDATA\Claude\claude_desktop_config.json
```

## 3. Inhalt einfügen

Siehe Datei:

```text
H:\SecondBrainAgent\SecondBrain-Agent\config\claude\claude_desktop_config_secondbrain.json
```

## 4. Claude Desktop neu starten

Claude komplett schließen und neu öffnen.

## 5. Test in Claude

```text
Nutze secondbrain und erstelle eine Testnotiz im Ordner 00_Inbox mit dem Titel MCP_Test.
```

## 6. Ergebnis prüfen

```text
H:\SecondBrainAgent\SecondBrain\00_Inbox\MCP_Test.md
```

## 7. Typische Befehle

```text
Merke dir diese Unterhaltung.
Erstelle daraus eine Projektnotiz.
Füge das zu Jarvis hinzu.
Suche alles zu Cisco EA.
Starte den SecondBrain OS Cycle.
```
