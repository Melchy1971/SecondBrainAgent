# SecondBrainAgent (Arbeitsverzeichnis)

Das Projekt liegt in `SecondBrain-Agent/`. Die massgebliche Dokumentation ist:

```text
SecondBrain-Agent\README.md
```

Alle Befehle laufen aus dem Projektordner:

```powershell
cd H:\SecondBrainAgent\SecondBrain-Agent
python launcher.py jarvis
```

`Jarvis.bat` bzw. die Startmenue-Verknuepfung "Jarvis" starten die native Desktop-App (primaere Oberflaeche seit v30.25). Das Web-HUD auf `http://127.0.0.1:8851` ist nur noch Kompatibilitaetsmodus (`python launcher.py hud`).

Hinweis: Dieser Root-README war bis v30.21 eine eigenstaendige Kopie der Projektdokumentation und ist mehrfach vom Projektstand abgewichen. Er wird bewusst nicht mehr inhaltlich gepflegt — Aenderungen gehoeren in `SecondBrain-Agent\README.md`.
